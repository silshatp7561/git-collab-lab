# 🗳️ Online Voting System using Blockchain

## 📌 Description
This project is a secure and transparent **Online Voting System** built using blockchain technology. It allows users to cast votes digitally while ensuring data integrity, security, and prevention of tampering.

The system leverages blockchain principles to store votes in an immutable structure, making the voting process trustworthy and reliable.

---

## 🚀 Features
- 🔐 User Registration & Login Authentication  
- 🗳️ Secure Online Voting  
- ⛓️ Blockchain-based vote storage  
- ❌ Prevention of duplicate voting  
- 📊 Transparent vote counting  
- 🧾 Tamper-proof records

## ⛓️ Blockchain Implementation
- Each vote is stored as a **block**
- Block contains:
  - Hashed voter ID  
  - Selected candidate  
  - Timestamp  
  - Previous block hash  
- Blocks are linked using cryptographic hashing
- Any modification breaks the chain → ensures security

---

## 🔐 Security Features
- Password hashing using Django authentication  
- Unique voter validation  
- Prevention of multiple votes  
- Blockchain ensures immutability  
- Secure session handling
