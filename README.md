# 🗳️ Online Voting System (Blockchain-Based)

## 📌 Project Overview

This project is a **secure online voting system** built using Django and enhanced with **blockchain technology** to ensure:

- 🔐 Data integrity  
- 🛡️ Security  
- 🔍 Transparency  

It provides a complete voting workflow for both **admin and voters**, from election creation to result analysis.

---

## 🚀 Features

- 🔑 User Authentication (Admin & Voter)
- 🧾 Voter Registration
- 🗳️ Election Creation & Management
- 👥 Candidate & Post Creation
- 📊 Live Results Display
- 🔗 Blockchain-secured Voting Records
- 🖥️ Admin Dashboard
- 👤 Voter Dashboard
- ⚙️ Django Admin Panel
